KVM的编译和安装
系统版本与硬件支持

楼主的内核版本：
vince67@miss:~$ uname -a
Linux miss 3.8.0-29-generic #42-Ubuntu SMP Tue Aug 13 23:12:18 UTC 2013 i686 i686 i686 GNU/Linux
硬件支持：
KVM需要硬件虚拟化扩展的支持，安装之前请先查看自己的cpu是不是支持扩展。
终端运行: $ grep -E '(vmx|svm)' /proc/cpuinfo
用于查看是否支持虚拟化扩展，如果你找到高亮的'vmx'(intel)或者'svm'(AMD)，说明是cpu是支持的。

KVM源代码下载

接下来我们要下载KVM的源码，我们在root/下建立一个kvm_demo的文件夹，进入后使用git clone命令下载.
$ git clone git://git.kernel.org/pub/scm/virt/kvm/kvm.git kvm.git
如果你的机器里没有安装git：
apt-get git
